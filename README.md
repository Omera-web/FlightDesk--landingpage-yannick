# FlightDesk — Landing page

Site statique multi-pages (HTML, CSS, JS).

## Pages

- `index.html` — Accueil (hero avec animation jet)
- `services.html` — Nos services
- `securite.html` — Vos données
- `equipe.html` — L'équipage
- `contact.html` — Contact

## Fichiers partagés

- `style.css` — Design system
- `main.js` — Navigation mobile et animations au scroll
- `jet-animation.html` — Animation canvas (ne pas modifier)

## Prévisualisation locale

```bash
python3 -m http.server 8080
```

Puis ouvrir `http://localhost:8080/index.html`
